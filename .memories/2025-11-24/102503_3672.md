Switched requires-python from >=3.14 to >=3.12 because PyTorch CUDA wheels for Windows only support up to 3.13, and torch-directml (for Intel Arc GPUs) only supports up to 3.12. Python 3.12 is the sweet spot for Windows GPU support across NVIDIA (CUDA), Intel (DirectML), and AMD (DirectML).
