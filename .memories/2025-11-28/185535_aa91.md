Diagnosed CI wheel test failure: undefined symbol le16toh. Root cause: tree-sitter grammars use le16toh (byte-order conversion) which isn't properly available in manylinux_2_17 (glibc 2.17). Fix: use manylinux_2_28 (glibc 2.28). Debug approach: added explicit error reporting instead of letting __init__.py swallow ImportError, which revealed the actual linker error.
